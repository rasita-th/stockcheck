# Public GitHub Upload Notes — Stock Radar V2.8

This version is public-GitHub safe.

## Before pushing

Make sure these files are not present:

```bash
find . -name "alpha_vantage_api_key.txt" -o -name ".env"
grep -R "YOUR_KEY_PREFIX" . --exclude-dir=.git
```

If either command finds your real key, remove it before pushing.

## Deploy model

V2.8 is built for GitHub Pages + GitHub Actions only:

- no Render/Railway/VPS required
- no Python backend running 24/7
- static JSON generated by GitHub Actions

## GitHub Pages setting

Go to:

```text
Settings → Pages → Build and deployment → Source → GitHub Actions
```

Then run these workflows in Actions:

1. `Update static fundamental data`
2. `Update technical data and deploy GitHub Pages`

## Alpha Vantage key

Do not add an Alpha Vantage key to the repo.

Users paste their own key in the Analyst Consensus tab. It is stored only in browser localStorage.
