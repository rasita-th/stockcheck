# Stock Timing Radar v6.8 — Mobile screener + FAB fix

Fixes:

- Mobile `+ New` screener now creates a new screener only; it no longer triggers the delete confirmation from long-press handling.
- Mobile portfolio tabs suppress accidental long-press delete on touch devices.
- Memo floating `+` button is separated from the notification bell so the two controls do not overlap.
- Scanner/Memo top mode buttons and screener tabs get safer mobile spacing.

Desktop behavior is unchanged: rename/delete/import remain available through their explicit buttons.
